Common spoofing Tactics:

  • Caller ID spoofing to appear legitimate
  • Creating urgency or fear (account locked, overdue arrest)
  • Requesting sensitive information for "verification"
  • Impersonating trusted organizations or government
  • Requests for gift cards or wire transfers

Protection Strategies:

  • Never give personal info to unsolicited callers
  • Hang up and call back using official numbers
  • Verify urgent requests through other channels
  • Let unknown callers go to voicemail
  • Report suspicious calls to authorities
